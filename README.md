## WooCommerce POS Email Invoice Gateway

### Instructions

1. Check [releases](https://github.com/wcpos/email-invoice-gateway/releases) for the latest version of the plugin.
2. Download the **woocommerce-pos-email-invoice-gateway.zip** file.
3. Install & activate the plugin via `WP Admin > Plugins > Add New > Upload Plugin`.
<img width="959" alt="Screenshot 2024-06-20 at 9 37 31 PM" src="https://github.com/wcpos/email-invoice-gateway/assets/639807/ebd29445-02df-4563-936a-1ef3ae743101">

4. Go to `WP Admin > WooCommerce POS > Settings > Checkout > enable` the WooCommerce POS Invoice Payment Gateway.
<img width="743" alt="Screenshot 2024-06-20 at 9 38 00 PM" src="https://github.com/wcpos/email-invoice-gateway/assets/639807/0cd46428-252a-4742-9613-7a98687746df">
